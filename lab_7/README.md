# Лабораторная работа №7

## Ансамблевые модели

Лабораторная работа по применению ансамблевых моделей машинного обучения к задаче классификации. Работа включает построение модели Random Forest для диагностики опухолей, настройку гиперпараметров и оценку качества:
- Применение ансамблевой модели к задаче бинарной классификации
- Подбор гиперпараметров и сравнение метрик качества

## Описание данных

#### Задача: Диагностика опухолей молочной железы
- **`breast-cancer-wisconsin.csv`** - данные о диагностике опухолей из набора Wisconsin Breast Cancer Dataset
  - Признаки: характеристики клеточных ядер (радиус, текстура, периметр, площадь и др.)
  - Целевая переменная: тип опухоли (доброкачественная/злокачественная)

Формат файла:
- Первая строка: заголовок столбцов
- Разделитель: запятая
- Кодировка: UTF-8
- Тип задачи: Классификация (бинарная)
- Количество образцов: 569
- Количество признаков: 30

## Задачи

#### Задача 1 (3 балла)
Построение и настройка ансамблевой модели:

1. Выбор задачи (бинарная классификация)
2. Выбор модели (Random Forest Classifier)
3. Подготовка данных (загрузка, разделение на train/test)
4. Выбор 1-2 гиперпараметров для настройки (n_estimators, max_depth)
5. Обучение модели на нескольких значениях гиперпараметров

#### Задача 2 (2 балла)
Оценка качества и анализ результатов:

1. Расчёт показателей качества (Accuracy, Precision, Recall, F1, ROC-AUC)
2. Сравнение с базовыми моделями (логистическая регрессия, дерево решений)
3. Визуализация зависимости качества от гиперпараметров
4. Формирование выводов об эффективности ансамблевой модели

## Основные функции sklearn

В работе используются:
- `sklearn.ensemble.RandomForestClassifier` - случайный лес для классификации
- `sklearn.model_selection.GridSearchCV` - поиск гиперпараметров
- `sklearn.model_selection.train_test_split` - разделение данных
- `sklearn.metrics.accuracy_score`, `f1_score`, `roc_auc_score` - метрики качества
- `sklearn.metrics.confusion_matrix` - матрица ошибок
- `pandas.read_csv()` - загрузка данных
