# Лабораторная работа №7

## Ансамблевые модели

Лабораторная работа по анализу данных с использованием ансамблевых моделей машинного обучения. Работа включает применение методов ансамблирования к задаче классификации или регрессии, настройку гиперпараметров и оценку качества:
- Выбор задачи и подходящей ансамблевой модели (Random Forest, Gradient Boosting)
- Подбор гиперпараметров и сравнение метрик качества с базовыми моделями

## Описание данных

#### Задача: Предсказание оттока клиентов
- **`telecom_churn.csv`** - данные о клиентах оператора сотовой связи
  - Признаки: демография, тариф, usage, услуги
  - Целевая переменная: отток (Churn)

Формат файла:
- Первая строка: заголовок столбцов
- Разделитель: запятая
- Кодировка: UTF-8
- Тип задачи: Классификация (бинарная)

## Задачи

#### Задача 1 (3 балла)
Построение и настройка ансамблевой модели:

1. Выбор задачи (классификация или регрессия)
2. Выбор модели (Случайный лес, Градиентный бустинг и др.)
3. Подготовка данных (загрузка, предобработка)
4. Выбор 1-2 гиперпараметров для настройки (например, `n_estimators`, `max_depth`)
5. Обучение модели на нескольких значениях гиперпараметров

#### Задача 2 (2 балла)
Оценка качества и анализ результатов:

1. Расчёт показателей качества (Accuracy, Precision, Recall, F1, ROC-AUC)
2. Сравнение с результатами базовых моделей (из предыдущих работ)
3. Визуализация зависимости качества от гиперпараметров
4. Формирование выводов о эффективности ансамблевой модели

## Основные функции sklearn

В работе используются:
- `sklearn.ensemble.RandomForestClassifier` - случайный лес
- `sklearn.ensemble.GradientBoostingClassifier` - градиентный бустинг
- `sklearn.model_selection.GridSearchCV` - поиск гиперпараметров
- `sklearn.metrics.accuracy_score`, `f1_score` - метрики качества
- `pandas.read_csv()` - загрузка данных
