# Лабораторная работа №1

## Использование библиотеки numpy

Лабораторная работа по анализу данных с использованием библиотек `numpy`, `scipy` и `matplotlib`. Работа включает два основных задания:
- Анализ данных о производстве и потреблении электроэнергии странами мира
- Построение и оценка полиномиальных моделей для прогнозирования прибыли

## Описание данных

#### Задача 1: Анализ электроэнергии
- **`global-electricity-generation.csv`** - данные о производстве электроэнергии странами с 1992 по 2021 год (млрд. кВт·ч)
- **`global-electricity-consumption.csv`** - данные о потреблении электроэнергии странами с 1992 по 2021 год (млрд. кВт·ч)

Формат файлов:
- Первая колонка: название страны
- Остальные колонки: данные по годам (1992-2021)
- Разделитель: запятая
- Первая строка: заголовок

#### Задача 2: Моделирование прибыли
- **`data2.csv`** - синтетические данные о зависимости прибыли от скидки
  - Первый столбец: размер скидки (%)
  - Второй столбец: прибыль

## Задачи

#### Задача 1 (5 баллов)
Анализ производства и потребления электроэнергии:

1. Загрузка данных из CSV-файлов
2. Построение массивов среднего производства и потребления за последние 5 лет
3. Аналитика:
   - 3.1. Суммарное потребление по годам
   - 3.2. Максимальное производство одной страной за год
   - 3.3. Страны с производством >500 млрд. кВт·ч
   - 3.4. Топ-10% стран по потреблению
   - 3.5. Страны с 10-кратным ростом производства (1992→2021)
   - 3.6. Страны с дефицитом энергии (потребили >100 млрд. кВт·ч больше, чем произвели)
   - 3.7. Страна с максимальным потреблением в 2020 году

#### Задача 2 (5 баллов)
Полиномиальная регрессия:

1. Построение системы линейных уравнений для полинома 2-й степени
2. Решение СЛУ и нахождение коэффициентов
3. Получение предсказанных значений
4. Визуализация исходных данных и модели
5. Расчёт RSS (Residual Sum of Squares)
6. Построение полинома 3-й степени и сравнение моделей
7. Прогнозирование прибыли при скидках 6% и 8%
