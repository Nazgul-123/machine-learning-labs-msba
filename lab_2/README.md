# Лабораторная работа №2

## Использование библиотеки pandas

Лабораторная работа по анализу данных с использованием библиотек `pandas`, `numpy`, `scipy` и `matplotlib`. Работа включает два основных задания:
- Анализ данных об участниках олимпийских игр
- Анализ оттока клиентов оператора сотовой связи

## Описание данных

#### Задача 1: Олимпийские игры
- **`athlete_events.csv`** - данные о спортсменах – участниках олимпийских игр
  - Основные признаки: ID, Name, Sex, Age, Height, Weight, Team, NOC, Games, Year, Season, City, Sport, Event, Medal

#### Задача 2: Отток клиентов телеком
- **`telecom_churn.csv`** - данные о клиентах оператора сотовой связи
  - Основные признаки: State, Area code, International plan, Total day/eve/night minutes/calls, Customer service calls, Churn

## Задачи

#### Задача 1 (5 баллов)
Анализ данных олимпийских игр:

1. Загрузка данных из CSV-файла
2. Анализ полноты данных (количество значений, отсутствующие данные)
3. Статистическая информация (среднее, отклонение, мин/макс, квартили) по возрасту, росту, весу
4. Аналитика:
   - 4.1. Самый молодой участник 1992 года (имя, дисциплина)
   - 4.2. Список всех видов спорта (уникальные значения)
   - 4.3. Средний рост теннисисток на играх 2000 года
   - 4.4. Золотые медали Китая в настольном теннисе (2008)
   - 4.5. Сравнение количества видов спорта (летние ОИ 2004 vs 1988)
   - 4.6. Гистограмма возраста мужчин-керлингистов (2014)
   - 4.7. Группировка по странам (Зимние ОИ 2006): медали и средний возраст
   - 4.8. Сводная таблица (pivot_table): медали по странам (Зимние ОИ 2006)

#### Задача 2 (5 баллов)
Анализ оттока клиентов (Churn Analysis):

1. Общая информация о датафрейме (info/describe), проверка на пропуски
2. Подсчет активных и потерянных клиентов (value_counts), проценты
3. Feature Engineering: добавление столбца «средняя продолжительность звонка», сортировка
4. Группировка по «Churn»: средняя продолжительность звонка
5. Группировка по «Churn»: среднее количество звонков в поддержку
6. Таблица сопряженности (crosstab): «Churn» vs «Customer service calls»
7. Таблица сопряженности (crosstab): «Churn» vs «International plan»
8. Прогнозирование оттока на основе правил, расчет ошибок первого и второго рода

## Основные функции pandas

В работе используются:
- `pandas.read_csv()` - загрузка CSV
- `DataFrame.info()`, `DataFrame.describe()` - информация о данных
- `DataFrame.value_counts()` - подсчет уникальных значений
- `DataFrame.groupby()` - группировка данных
- `DataFrame.pivot_table()` - сводные таблицы
- `pandas.crosstab()` - таблицы сопряженности
- `matplotlib.pyplot.hist()` - построение гистограмм
