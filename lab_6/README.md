## Описание данных

#### Задача: Кластеризация звёзд
- **`stars.csv`** - синтетические данные о расположении звёзд на карте
  - **x** - координата звезды по оси X (условные единицы)
  - **y** - координата звезды по оси Y (условные единицы)

Данные содержат информацию о звёздах трёх кластеров. Каждый кластер – это набор звёзд (точек), лежащий внутри круга некоторого радиуса. Каждая звезда принадлежит ровно одному кластеру. Расстояние между точками измеряется как евклидово расстояние.

## Задачи

#### Основная часть (4 балла)

1. Загрузка данных из CSV-файла
2. Проверка правильности загрузки данных и их типа
3. Кластеризация алгоритмом KMeans:
   - Количество кластеров: 3 (исходя из описания данных)
   - Начальный выбор центроидов: случайный
   - Получение разбиения на кластеры
4. Построение точечного графика:
   - Ось X: x-координаты звёзд
   - Ось Y: y-координаты звёзд
   - Цвет точек: соответствует кластеру
5. Расчёт метрики качества «силуэт» (silhouette score) для полученного разбиения
6. Эксперимент с другим количеством кластеров (например, 5):
   - Построение модели
   - Расчёт silhouette score
   - Сравнение с исходным результатом

#### Дополнительное задание (+1 балл)

7. Самостоятельная реализация алгоритма k-means:
   - Инициализация центроидов
   - Назначение точек кластерам
   - Пересчёт центроидов
   - Итерации до сходимости
8. Сравнение результатов своей реализации со стандартной (sklearn)

## Основные функции и библиотеки

В работе используются:
- `pandas.read_csv()` - загрузка CSV
- `DataFrame.info()`, `DataFrame.head()` - информация о данных
- `sklearn.cluster.KMeans()` - алгоритм кластеризации KMeans
- `sklearn.metrics.silhouette_score()` - метрика силуэт
- `matplotlib.pyplot.scatter()` - точечная диаграмма
- `numpy` - работа с массивами и вычисления

## Метрики качества

| Метрика | Описание |
|---------|----------|
| Silhouette Score (Силуэт) | Мера качества кластеризации от -1 до 1 (чем ближе к 1 - лучше). Показывает, насколько точка похожа на свой кластер по сравнению с другими кластерами |

## Параметры KMeans

| Параметр | Значение | Описание |
|---------|----------|----------|
| n_clusters | 3 (или 5) | Количество кластеров |
| init | 'random' | Случайная инициализация центроидов |
| n_init | 10 | Количество запусков алгоритма |
| max_iter | 300 | Максимальное количество итераций |
