## Описание данных

#### Задача: Анализ погодных данных
- **`weather1.csv`** - данные о погоде в Перми
  - **Местное время в Перми** - дата и время наблюдения
  - **T** - температура воздуха в градусах Цельсия
  - **P** - атмосферное давление в мм.рт.ст.
  - **U** - относительная влажность в %
  - **Ff** - скорость ветра в м/с
  - **N** - облачность
  - **H** - высота основания облаков, м
  - **VV** - горизонтальная дальность видимости в км

## Задачи

#### Загрузка данных (обязательно)
1. Загрузка данных из CSV-файла
2. Выбор только необходимых столбцов (время, T, P, U, Ff, N, H, VV)

#### Визуализация (10 баллов)

1. **(1 балл)** Точечная диаграмма (scatter plot) по признакам температуры и относительной влажности

2. **(1 балл)** Цветовое выделение точек на диаграмме в зависимости от облачности:
   - Синим – облачность 100%
   - Красным – все остальные значения
   - Полупрозрачность для точек на переднем плане

3. **(1 балл)** Линейная диаграмма изменения температуры от местного времени:
   - Подписи на оси времени должны быть различимы
   - Подписывать только дни или с интервалом

4. **(2 балла)** Столбчатая диаграмма среднемесячной температуры:
   - Вычисление номера месяца из столбца времени
   - Группировка данных по месяцу
   - Вертикальная столбчатая диаграмма

5. **(0,5 балла)** Ленточная диаграмма (горизонтальная) количества наблюдений для каждого варианта облачности

6. **(0,5 балла)** Гистограмма частот для температуры (10 диапазонов)

7. **(2 балла)** Boxplot для атмосферного давления по 3 группам видимости:
   - Группа 1: видимость < 5 км
   - Группа 2: видимость от 5 до 15 км (включительно)
   - Группа 3: видимость > 15 км

8. **(1 балл)** Круговая диаграмма для высоты основания облаков:
   - Группы < 5% объединить в категорию «Другое»

9. **(1 балл)** Сводная таблица (pivot_table) и тепловая карта (heatmap):
   - Строки: скорость ветра
   - Столбцы: облачность
   - Значения: средняя температура

## Основные функции и библиотеки

В работе используются:
- `pandas.read_csv()` - загрузка CSV с выбором столбцов
- `matplotlib.pyplot.scatter()` - точечная диаграмма
- `matplotlib.pyplot.plot()` - линейный график
- `matplotlib.pyplot.bar()` / `barh()` - столбчатые диаграммы
- `matplotlib.pyplot.hist()` - гистограмма
- `matplotlib.pyplot.boxplot()` - ящик с усами
- `matplotlib.pyplot.pie()` - круговая диаграмма
- `seaborn.heatmap()` - тепловая карта
- `pandas.pivot_table()` - сводные таблицы
- `pandas.groupby()` - группировка данных

## Требования к визуализации
