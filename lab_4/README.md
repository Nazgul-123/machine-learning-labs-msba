## Описание данных

#### Задача: Прогнозирование цены недвижимости
- **`boston.csv`** - данные о недвижимости в различных районах Бостона

| Признак | Описание |
|---------|----------|
| CRIM | Уровень преступности |
| ZN | Доля жилых земель (>25 000 кв.футов) |
| INDUS | Доля площадей, не связанных с розничной торговлей |
| CHAS | Наличие реки (1/0) |
| NOX | Качество воздуха (концентрация оксидов азота) |
| RM | Среднее количество комнат в доме |
| AGE | Доля помещений, построенных до 1940 года |
| DIS | Взвешенные расстояния до центров занятости |
| RAD | Индекс доступности автомагистралей |
| TAX | Ставка налога на 10 000 долларов |
| PTRATIO | Соотношение учеников и учителей |
| B | Нормированная доля афроамериканцев |
| LSTAT | Процент населения с низким статусом |
| MEDV | Медианная цена недвижимости (тыс.$) - целевой признак |

## Задачи

#### Основная часть (6 баллов)

1. Загрузка данных из CSV-файла
2. Проверка типов данных (все признаки должны быть числовыми)
3. Проверка и заполнение пропущенных данных (медианой)
4. Расчёт корреляционной матрицы для всех пар признаков
5. Построение тепловой карты (heatmap) корреляций
6. Выбор 4-6 признаков с наибольшей корреляцией с целевым признаком
7. Построение scatter plot для выбранных признаков vs целевой переменной
8. Визуальная оценка наличия зависимости
9. Формирование матрицы признаков и целевой переменной
10. Разбиение на обучающую и тестовую выборки (80:20)
11. Обучение линейной регрессии (Linear Regression)
12. Получение прогнозов на train и test выборках
13. Расчёт метрик: R² и RMSE для обеих выборок

#### Дополнительные задания (4 балла)

14. Построение boxplot для целевого признака (MEDV), определение выбросов
15. Фильтрация выбросов, переобучение модели, сравнение метрик
16. Обучение гребневой регрессии (Ridge), расчёт метрик
17. Построение полиномиальной регрессии (3-й степени), сравнение всех моделей

## Основные функции и библиотеки

В работе используются:
- `pandas.read_csv()` - загрузка CSV
- `DataFrame.info()`, `DataFrame.describe()` - информация о данных
- `DataFrame.corr()` - корреляционная матрица
- `DataFrame.fillna()` - заполнение пропусков
- `sklearn.model_selection.train_test_split()` - разбиение выборки
- `sklearn.linear_model.LinearRegression()` - линейная регрессия
- `sklearn.linear_model.Ridge()` - гребневая регрессия
- `sklearn.preprocessing.PolynomialFeatures()` - полиномиальные признаки
- `sklearn.metrics.r2_score()`, `mean_squared_error()` - метрики качества
- `seaborn.heatmap()` - тепловая карта
- `matplotlib.pyplot.boxplot()` - ящик с усами
- `matplotlib.pyplot.scatter()` - точечная диаграмма

## Метрики качества

| Метрика | Описание |
|---------|----------|
| R² (коэффициент детерминации) | Доля дисперсии, объяснённая моделью (0-1, чем ближе к 1 - лучше) |
| RMSE (корень из MSE) | Среднеквадратичная ошибка (чем меньше - лучше) |
